# nodeditor [WIP]

Simple console text editor written in 100% JavaScript.

## Requirements

- `node` v6.0.0+
- `npm`

## Installation

`npm i -g nodeditor`

## Usage

Start from command line:

```sh
nodeditor [options]
```

## CLI Options

- `-V`, `--version` -- output the version number

- `-h`, `--help` -- output usage information
